例子02

全新揽胜极光搭载了智能四驱系统和第二代智能主动动力传输系统，其传动系统切换功能可实现四驱到两驱的切换，从而有效提升能效表现。

  1. POS语法标注 
    [全新][揽胜极光][搭载][了][智能][四驱][系统][和][第二代][智能][主动][动力][传输][系统]，[其][传动][系统][切换][功能][可][实现][四驱][到][两驱][的][切换]，[从而][有效][提升][能效][表现]。
  2. 成分标注 
    [NP 全新揽胜极光][V 搭载了][NP 智能四驱系统]和[NP 第二代智能主动动力传输系统]，其[NP 传动系统切换功能]可[V 实现][VP 四驱到两驱的切换]，从而[ADJ 有效][VP 提升}[NP 能效表现]。
  3. 语义角色标注 
    全新揽胜极光搭载了智能四驱系统和第二代智能主动动力传输系统；
全新揽胜极光传统系统切换功能可实现四驱到两驱的切换；
【上述行为】从而有效提升能效表现。
    ```yaml CAR: &subject_car Model: 揽胜极光 Adj: 全新

EMPLOY/CONTAINS/HAS: 搭载 &employ Subject: *subject_car Object: 智能四驱系统，第二代智能主动动力传输系统 &trans

CAPABLE/ENABLES: 可实现 &capable Subject: subject_car Attribute: trans Attribute: 切换功能 Object: 四驱到两区的切换

CHANGE: 提升 &change Adj: 有效 Object: 能效表现

IMPLICATION: Antecedent: capable Concequent: change

```

例子05

风尚版及舒适版分别采用16寸与17寸铝合金轮圈，而豪华版与旗舰版升级则是到了18寸的战斧式轮圈。 1. POS

2. 成分 
[NP 风尚版及舒适版][VP 分别采用][NP 16寸与17寸铝合金轮圈]，[C 而][NP 豪华版与旗舰版][VP 升级则是到了][NP 18寸的战斧式轮圈]。
3. 语义角色标注 ```yaml EMPLOY: 采用 Subject.1: CAR: default Edition: 风尚版 Object.1: 16寸铝合金轮圈 Subject.2: CAR: default Edition: 舒适版 Object.2: 17寸铝合金轮圈

CHANGE: 升级 Subject: CAR: default Edition: 豪华版，旗舰版 Object: 18寸的战斧式轮圈 ```

这里的主要问题是遇到一一对应描述的语句时，是否通过Subject.n与Object.n对应的方式来处理。
